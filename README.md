# Frogger-Style Tutorial Assets (Godot)

This repository contains **only the visual and audio assets** used in a **Frogger-style game tutorial** built with **Godot**.


---

## What’s Included

- **`/Assets`**
  - Sprites used throughout the tutorial
  - Sound effects used in the tutorial


---

## What’s NOT Included

- No Godot project files  
- No `.tscn` scenes  
- No scripts or gameplay logic  
- No movement, scoring, or win/lose systems  

All setup and implementation happens **step by step in the tutorial**.
